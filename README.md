# cw-yiyan-plugin

![image](https://github.com/user-attachments/assets/73f15952-35e6-48c0-9d45-b4544c2c5a9b)

## 简介
一个用于展示一言的插件。每天分享一句话，激励大家学习。

## 使用说明
您可以在主程序的 `SmoothScrollArea` 方法中修改 `is_infinite` 参数来控制是否启用无限循环。
